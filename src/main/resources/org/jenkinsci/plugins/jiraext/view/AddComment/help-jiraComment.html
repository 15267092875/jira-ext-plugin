Commit messages are assumed to have the Jira ticket they are for as the first word of the ticket.

For each of those Jira tickets then, post a comment.

Define the comment here. Environment variables can be passed with the $ character, i.e: $BUILD_ID

In addition, The following properties are made available:

  • $AUTHOR - the author of the changeset, i.e dalvizu
  • $COMMIT_ID - The commit id
  • $COMMIT_DATE - Friendly format of the commit's date
  • $COMMIT_MESSAGE - The message of the commit
